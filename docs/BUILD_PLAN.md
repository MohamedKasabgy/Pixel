# Build Plan — 3 Week MVP

## Week 1 — Foundation + Visual Demo

- WordPress install and environment setup.
- Activate WooCommerce.
- Install Pixel theme and Pixel Core plugin.
- Build visual pages:
  - Home
  - Products / Categories
  - Product Details
  - Request Quote
  - Upload Artwork
  - Client Dashboard
  - Order Tracking
  - Cart
  - Checkout
  - Static support/policy pages
- Create the initial 10–15 product catalog.
- Configure brand colors, logo, typography, header, footer.

## Week 2 — Core Business Flow

- Product options and variable products.
- Quote request form submission into admin.
- Artwork upload with validation.
- WooCommerce checkout and payment gateway setup.
- Order status labels for print production.
- Customer portal shortcodes and order views.
- Coupon/discount setup.

## Week 3 — Polish + Admin + Launch Prep

- Admin/staff workflow polish.
- Invoice view and downloadable invoice template.
- Shipping/delivery/pickup pages.
- QA on desktop/mobile.
- Basic SEO, analytics, caching, backups, security.
- Content cleanup and client review.
- Launch checklist.
